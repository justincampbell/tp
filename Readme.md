```sh
gem install tp
tp slides.md
```

* H,J,K,L
* W,A,S,D
* Q to quit

# Paragraphs

Short paragraphs are centered

# Paragraphs

Sometimes paragraphs are really really long and might wrap a few lines. It does its best to wrap them logically. Also, they stay left-aligned.
